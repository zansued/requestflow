# RequestFlow

RequestFlow é um sistema interno de solicitações e aprovações baseado na web, projetado para organizações, permitindo que os usuários enviem solicitações e que os gerentes aprovem, rejeitem ou solicitem ajustes, com audit trails completos.

## Pré-requisitos

- Node.js
- npm

## Instruções de Instalação

```bash
git clone https://github.com/seu_usuario/requestflow.git
cd requestflow
npm install
npm run dev
```

## Scripts Disponíveis

- `dev` - Executa o aplicativo em modo de desenvolvimento.
- `build` - Cria uma versão otimizada para produção.
- `preview` - Visualiza a versão de produção localmente.
